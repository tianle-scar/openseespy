## 分析内容

本项目建立一榀五层三跨钢筋混凝土平面框架非线性有限元模型，采用纤维截面 `forceBeamColumn` 单元模拟梁柱构件，完成以下分析：

1. 竖向荷载作用下的内力、变形分析。
2. 频遇地震作用下的底部剪力法分析。
3. 非线性动力时程分析。
4. 静力 Pushover 分析。
5. 截面材料本构曲线绘制。
6. 塑性铰定义、判别、统计和可视化。

## 主要结果

- 基本周期：`T1 = 1.037 s`
- 竖向荷载总反力：`5053.6 kN`
- 频遇地震等效基底剪力：`451.4 kN`
- 非线性时程最大屋顶位移：`66.85 mm`
- Pushover 最大基底剪力：`909.2 kN`
- Pushover 等效屈服位移：`133.5 mm`
- Pushover 等效屈服力：`839.1 kN`
- 延性系数：`5.48`
- 推覆最终状态塑性或严重塑性铰数量：`45`

## 程序运行

建议使用 Python 3.9 及以上版本。运行前安装依赖：

```bash
pip install openseespy numpy matplotlib scipy
```

运行程序：

```bash
python 2025210070-付天乐.py
```

程序运行后会在当前目录下生成或刷新 `analysis_results/` 文件夹，包含：

- 结构信息图、模型图、振型图；
- 竖向荷载、频遇地震和推覆分析内力图；
- 人工地震动与时程响应图；
- Pushover 曲线、等效屈服点和刚度退化图；
- 塑性铰分布图；
- 构件内力 CSV、时程响应 CSV、塑性铰 JSON/CSV。

## 塑性铰判别方法

程序采用构件端弯矩需求比定义塑性铰：

```text
DCR = |M_end| / My
```

其中 `M_end` 为构件端弯矩，`My` 为按矩形钢筋混凝土截面受压区平衡估算的屈服弯矩。状态划分如下：

| 状态 | 判别条件 |
|---|---|
| 弹性 | `DCR < 0.80` |
| 接近屈服 | `0.80 <= DCR < 1.00` |
| 塑性铰 | `1.00 <= DCR < 1.30` |
| 严重塑性 | `DCR >= 1.30` |

该判别用于结构薄弱部位识别和可视化展示，详细转角能力仍应结合截面分析或规范限值进一步复核。
